# 健身核心理论手册 (V3.2 - 饮食全解析版)

**核心：线性周期、经验化饮食、全器械三分化与专项动作细节**

## 第一部分：训练理论——短期线性计划 (2个月周期)

基于最新实战模型，本阶段采用“2个月短期线性推进”策略：

### 1. 负荷管理规则

- **起始重量：** 从目前 1RM 的 70% 开始。
- **推进速度：** 动作标准的前提下，每周按比例增加。
- **频率策略：** 每周训练 2 次（1 次正常推进，1 次为 70% 重量轻训课）。
- **减载 (Deload) 触发器：** 连续 3 天训练热情下滑或睡眠困难；计划内固定减载。

### 2. 周期强度阶梯表 (参考)

| **阶段**   | **组次数**   | **强度 (1RM%)** | **目标**     |
| ---------- | ------------ | --------------- | ------------ |
| **初期**   | 5×10 -> 4×10 | 70% - 72.5%     | 建立容量基准 |
| **中期**   | 4×8 -> 3×8   | 75% - 77.5%     | 进阶肥大     |
| **减载周** | 3×8          | 降重 20%        | 消除疲劳     |
| **冲刺期** | 5×5 -> 3×5   | 80% - 87.5%     | 力量爆发     |
| **极限期** | 3×3 -> 1×2   | 90% - 95%       | 冲击新 1RM   |

## 第二部分：饮食理论——经验化模型 (Empirical Diet)

### 1. 基础配额模型 (g/kg 体重)

| **阶段**          | **碳水 (变量)**   | **蛋白质 (常量)** | **脂肪 (底线)** |
| ----------------- | ----------------- | ----------------- | --------------- |
| **增肌期 (男)**   | 3.5 - 4.5g        | 1.5 - 2g          | 1g              |
| **减脂期 (男)**   | 2 - 3g (初期递减) | 1.5g              | 0.8g            |
| **BMI > 28 群体** | 2.5g (起步)       | 1.2g              | 0.6g            |
| **BMI > 32 群体** | 2.0g (起步)       | 1.0g              | 0.5g            |

### 2. 动态调整办法 (以 2 周为周期观察)

- **增肌：** 体重能增约 1% 则不调整。长太快减碳水，长太慢加碳水，增减尺度约为 **0.5g/kg**。
- **减脂：** 体重能减约 2% 则不调整。掉太慢减碳水，掉太快加碳水。
- **重算规则：** 增肌期基本不重算；减脂期体重降低 5-10kg 可重算一次。

### 3. 训练日分餐方案 (2242法则)

*核心：碳水跟着训练走。*

| **餐次**      | **碳水分配** | **蛋白质** | **脂肪** | **场景与建议**                            |
| ------------- | ------------ | ---------- | -------- | ----------------------------------------- |
| **1. 早餐**   | 全天 20%     | 不严格     | 不严格   | 食堂主食 + 2个鸡蛋 + 牛奶                 |
| **2. 其他餐** | 全天 20%     | 不严格     | 不严格   | 午餐：主食 + 瘦肉 + 蔬菜多吃              |
| **3. 练前餐** | 全天 20%     | 可吃可不吃 | 一般不吃 | 练前 10-30min：馒头/面包/香蕉/快碳饮料    |
| **4. 练后餐** | 全天 40%     | 30-50g     | 20g 内   | 练后 30min 内：快碳 + 瘦肉 + 蔬菜少吃后吃 |

- **变式：** 若晚饭后训练，则晚饭为练前餐 (20% 碳水)，夜宵为练后餐 (40% 大餐)。
- **减脂微调：** 随着碳水减少，将 2242 比例压缩为 3052 (即取消其他餐碳水)。

### 4. 休息日分餐方案 (平稳法则)

- **总量降低：** 碳水比训练日降低约 **0.5g/kg** (增肌期 2.5-3g，减脂期 1.5-2g)。
- **构成均摊：** 各餐碳、蛋、脂大概均摊，不做碳水集中。
- **效果：** 稳定全天胰岛素水平，强效抑制脂肪合成。

## 第三部分：三分化进阶计划与动作解析

### **Day A: 拉 (Pull) —— “焚决”背部专项**

1. **[激活]** 祈祷式拉伸 + 颈后臂屈伸：手肘搭凳重心后推；体会肩胛骨滑动感。
2. **[垂直拉/厚度]** 反手高位下拉：握距同肩宽，肘部向内收，侧重下背阔肌。
3. **[垂直拉/宽度]** 对握高位下拉：身体保持刚性不后仰，侧重上背阔与大圆肌。
4. **[水平拉/厚度]** 窄对握绳索划船：脊柱中立，先预发力，感受下背收紧。
5. **[水平拉/细节]** 开肘划船：以肩胛后缩为主导，打造背部细节。
6. **[收尾/泵感]** 绳索直臂下压：手腕扣紧肘微抬，有控制下压，榨干背阔。
7. **[二头]** 杠铃弯举 + 锤式弯举。

### **Day B: 推 (Push) —— 胸肩增宽专项**

1. **[胸部主项]** 杠铃卧推：(1RM 基准 32kg)，收紧肩胛。
2. **[上胸强化]** 上斜哑铃卧推：3 × 10-12 (解决锁骨处饱满度)。
3. **[肩部主项]** 哑铃推举：4 × 12。
4. **[中束/肩宽]** 侧平举：4 × 15-20 (高次数泵感)。
5. **[三头]** 绳索下压 + 仰卧臂屈伸。

### **Day C: 蹲 (Legs) —— 腿部与核心全解析**

1. **[下肢主项]** 哈克深蹲 (基准 80kg)。
2. **[后链/臀腿]** 罗马尼亚硬拉 (RDL) (1RM 60kg)：髋主导，严禁圆背。
3. **[核心核心]** 反向山羊挺身：核心等长收缩，向上伸展而非卷腹。
4. **[核心刚性]** 悬垂举腿：使用辅助带，保持腹肌持续紧张，控制下放。
5. **[腹肌孤立]** 跪姿卷腹：髋部为轴，动作行程短。
6. **[提醒]** 练腹腰疼应降阶做平板支撑；清晰腹肌靠体脂率（饮食）。
